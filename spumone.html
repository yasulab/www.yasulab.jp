SPUMONE

SPUMONE (Software Processing Unit, Multiplexing ONE into two) とは, シングルコアおよびマルチコア CPU 上で動作する CPU 仮想化層のことで, 単一の物理 CPU から複数の仮想 CPU を生成し, ゲスト OS に提供することが出来る.

主に, Spinlock による Lock Holder Preemption の原因を解明し, SPUMONE のパフォーマンス向上することを努めた.

SPUMONEプロジェクトは, 未踏プロジェクトに２回採択されている.
